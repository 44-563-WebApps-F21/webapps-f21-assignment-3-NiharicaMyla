The definition of the area of a polygon is the measure of the area that is enclosed by it. As polygons are closed plane shapes, thus, the area of a polygon is the space that is occupied by it in a two-dimensional plane and always expressed in square units.

Quick link for algorithm description!! 

        double area(const vector& fig) {
            double res = 0;
            for (unsigned i = 0; i < fig.size(); i++) {
                point p = i ? fig[i - 1] : fig.back();
                point q = fig[i];
                res += (p.x - q.x) * (p.y + q.y);
            }
            return fabs(res) / 2;
        }
